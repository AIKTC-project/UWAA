# blog
A CMS(Content Management System) at scale.

# Tech stack
Frontend: HTML5, [CSS](https://sass-lang.com), [Svelte.js](https://svelte.dev)

Backend API: [Golang](https://go.dev)

Database: [PostgreSQL](https://postgresql.org), [Redis](https://redis.io)
